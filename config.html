cmiputil.config module¶

Maintain config file for cmiputil package.

Config file for this package is so called ‘INI file’, handled by python standard configparser module.

Default name of config file is cmiputil.conf, and searched in $HOME then current directory. If file is specified to the constructor, that is read and override the precedence.

“Default” section name is conf_section and the key=value pair is conf_default.

You can create sample(default) conf file by createSampleConf, collect default configuration of each module by getDefaultConf() in each module and read_dict(), then writeConf() to write the file.

class config.Conf(file='')[source]¶

Bases: configparser.ConfigParser

Parameters

file (str or path-like) – config file

If file is None, no conf file is read and blank instance is created. If you want only default conf files, set file="".

setDefaultSection()[source]¶

Set “default” section.

writeConf(fname, overwrite=False)[source]¶

Write current attributes to the fname.

You have to set configurations for each module via, for example, self.read_dict().

Parameters

  • fname (str or path-like) – file to be written

  • overwrite (bool) – force overwrite

Examples

>>> from cmiputil import esgfsearch
>>> conf = Conf('')
>>> conf.setDefaultSection()
>>> d = esgfsearch.getDefaultConf()
>>> conf.read_dict(d)
>>> conf.writeConf('/tmp/cmiputil.conf', overwrite=True)
