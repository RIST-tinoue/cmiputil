cmiputil.esgfsearch module¶

Search by ESGF RESTful API, get and open multiple datafiles, create meta info.

See https://earthsystemcog.org/projects/cog/esgf_search_restful_api for the detail ESGF RESTful API.

This version uses siphon and xarray.

esgfsearch.getCatURLs(params=None, base_url=None)[source]¶

Using ESGF RESTful API, get URLs for OPeNDAP TDS catalog.

Parameters

  • fields (dict) – keyword parameters and facet parameters.

  • base_url – base URL of the ESGF search service.

Raises

NotFoundError – raised if no catalog found.

Returns

list of str – TDS catalog URLs found.

If base_url is None, search_service + service_type is used.

field is to update (use update() method of python dict) to fields_defaults. You can/must set it via setDefaultFields() beforehand.

esgfsearch.getDataset(url, aggregate=True, netcdf=False)[source]¶

From given URL of TDS catalog, return xarray or netCDF4 dataset.

A URL of TDS catalog is obtained by, for example, getCatURLs().

If aggregate is True, access aggregated dataset, else access all of files listed in the catalog.

Parameters

  • url (str) – url of TDS catalog

  • aggregate (bool) – if True, access aggregate link in TDS catalog.

  • netcdf (bool) – if True return netCDF4’s dataset, instead of xarray’s.

Returns

xarray.dataset() or xarray.mfdataset() or netCDF4.Dataset() or netCDF4.MFDataset() or None: opened dataset

esgfsearch.getLocalPath(fields, base_dir=None)[source]¶

Search local data directory for CMIP6 data.

Using same interface with getCatURLs()

Parameters

  • fields (dict) – keyword parameters and facet parameters.

  • base_url – base path for local data directory.

Raises

NotFoundError – raised if no paths found.

Returns

list of path-like – paths found to match given facets.

field is initialized by keyword_defautls and facet_defaults.

Example

>>> params_update = {'source_id': 'MIROC6', 'experiment_id': 'historical',
...    'variant_label': 'r1i1p1f1', 'variable': 'tas', }
>>> params = fields_default
>>> params.update(params_update)
>>> str(getLocalPath(params, base_dir='~/Data/'))
'~/Data/CMIP6/*/*/MIROC6/historical/r1i1p1f1/Amon/*/*/*/*_Amon_MIROC6_historical_r1i1p1f1_*_*.nc'
