Usage¶

draw-timeseries.py¶

Draw time-series of CMIP6 data, searched via RESTful API and accessed by OPeNDAP.

This is just a “Proof-of-concept” for accessing CMIP6/ESGF data via cmiputil. Process after opening dataset is a sloppy and dirty hack. You should implemnt properly by yourself.

usage: draw-timeseries.py [-h] [-d] [-c CONFFILE] [-l] [params [params ...]]
