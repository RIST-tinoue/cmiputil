draw-timeseries program¶

Description¶

Usage¶

Draw time-series of CMIP6 data, searched via RESTful API and accessed by OPeNDAP.

usage: draw-timeseries [-h] [-c CONFFILE] [params [params ...]]

Positional Arguments¶

params

key=value series of keyword/facet parameters

Named Arguments¶

-c, --conffile

config file

This is just a “Proof-of-concept” for accessing CMIP6/ESGF data via cmiputil. So process after opening dataset is a sloppy and dirty hack. You should implemnt properly by yourself.
