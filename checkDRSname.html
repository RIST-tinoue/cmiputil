checkDRSname program¶

Description¶

Check file/directory name is valid as DRS.

Usage¶

Check file/directory name is valid as DRS.

If (one of) given file(s) is(are) invalid, exit with code 1. If verbose (-v), the reason is also shown.

usage: checkDRSname [-h] [-v] [-q] nc_file [nc_file ...]

Positional Arguments¶

nc_file

Named Arguments¶

-v, --verbose

Default: False

-q, --quiet

Default: False
